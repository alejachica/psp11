Instrucciones para obtener el repositorio desde github:
Hacer clone al proyecto con la siguiente URL: https://github.com/alejachica/psp11.git
usuario: alejachica

Para ejecutar el programa localmente, ejecutar el siguiente comando:
mvn package
java -cp target/psp11-1.0-SNAPSHOT.jar edu.uniandes.ecos.psp11.vista.VistaConsola

Para ejecutar el progama desde Heroku, ingresar a la siguiente URL:
https://psp11-alejachica.herokuapp.com/
